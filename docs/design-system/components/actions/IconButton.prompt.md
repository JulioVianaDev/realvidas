Icon-only button sharing Button's variants. Use `round` for the floating WhatsApp/call bubbles. Always pass `label` for accessibility.

```jsx
<IconButton icon={<Icon name="whatsapp" />} label="WhatsApp" variant="whatsapp" round size="lg" />
<IconButton icon={<Icon name="search" />} label="Buscar" variant="ghost" />
```

Variants: `primary` `secondary` `accent` `whatsapp` `outline` `ghost` `soft`. Sizes: `sm` `md` `lg`.
